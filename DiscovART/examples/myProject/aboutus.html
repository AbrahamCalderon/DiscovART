What is DisovART?

User Interface (UI) Design emphases on anticipating user needs and ensuring that interfaces consist of elements that are easily accessible, understandable, and useful in facilitating those needs. Consistency, simplicity, and a useful page layout are three major UI components the DisovART site incorporates in delivering a dynamic user interface.

DiscovART is a simple user friendly website with a primary goal of the utilizing HTML5 capabilities for developing rich user interfaces. DiscovART allows visitors to browse, share, and create simple art in the most efficient way possible via HTML5 and JavaScript technologies.
